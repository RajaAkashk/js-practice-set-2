Check Anagrams: Declare two variables word1 and word2 and assign them string values. Write a function to check if word1 is an anagram of word2. Print whether they are anagrams or not. Remove Characters: Declare a variable text and assign it a string value. Write a function to remove all occurrences of a specific character from the string. Print the modified text. Check Palindromic Sentence: Declare a variable sentence and assign it a string value containing a sentence. Write a function to check if the sentence is palindromic (reads the same forwards and backwards, ignoring spaces and punctuation). Print whether the sentence is palindromic or not. Split and Join: Declare a variable sentence and assign it a string value containing words separated by spaces. Write a function to split the sentence into an array of words, then join them back into a single string with dashes (-) between each word. Print the joined string. Find Longest Palindromic Substring: Declare a variable text and assign it a string value. Write a function to find the longest palindromic substring within the text. Print the longest palindromic substring.
